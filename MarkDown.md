# Инструкция для работы с Markdown

## Выделение текса

Чтобы выделить текст курсивом необходимо обрамить его (*), напримет *вот так*.
Чтобы выделить текст полужирным, необходимо выделить его двойными звездосками (**). Например **вот так**.
